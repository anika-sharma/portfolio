Mineral Project

The Project

A sleek monochromatic e-Commerce website, The Mineral Project is unique in many ways. To start with it is an enquiry only website for the products that are available, its simplicity depends heavily up on the vibrancy of the product images. A fully responsive framework with a vertical navigation bar and minimalistic features, it aims at providing smooth transitions within the website.
